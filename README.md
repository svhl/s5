## How to use

### C:

`cc [filename].c` or `cc [filename].c -o [name]`\
`./a.out` or `./[name].out`

### SQL:

Use any software that supports Oracle SQL. Describe tables and list rows after making changes to tables.

### MASM:

`mount c c:\8086\MASM`\
`c:`\
`masm [filename].asm`\
`link [filename].obj`\
`[filename]`
